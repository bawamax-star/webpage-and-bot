Kali linux help

  • Kali Linux is one of the best security based OS system. Ethical Hackers mostly use this form of linux.

  • Kali Linux contains set of tools divided by the categories.

  • It's an open source and it's main page is https://www.kali.org

  • You can download kali linux from https://www.kali.org/downloads/

If you are a kali linux user, then you will need these type of commands.

  • $ sudo apt-get update - This command will update your system.

  • $ sudo apt-get upgrade - This command will upgrade your system to the newest format of "kali linux".

  • $ sudo apt-get install wine - If you want to install windows formatted files on Linux, then you will need this.

  • $ sudo apt-get install gdebi-core - .deb package insataller. This will help you to install any application.

  • If you want more information go and visit kali linux help.
